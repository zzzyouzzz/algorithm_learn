# 二、核心基础算法（面试高频）
## 1. 排序算法
### （1）冒泡、选择、插入（理解原理）
- **冒泡排序**
  - 思想：相邻元素两两比较，大的往后“冒”
  - 时间：O(n²)，稳定
  - 用途：理解排序思想，面试很少手写
- **选择排序**
  - 思想：每次选最小/最大放到已排序区末尾
  - 时间：O(n²)，不稳定
- **插入排序**
  - 思想：像打牌一样，逐个插入到前面有序区
  - 时间：O(n²)，对近乎有序数组效率高，稳定

### （2）快速排序、归并排序（必须手写）
- **快速排序**
  - 思想：选基准 pivot，分区（小左大右），递归
  - 时间：平均 O(n log n)，最坏 O(n²)
  - 考点：手写、partition 过程、优化（三数取中、三向切分）
- **归并排序**
  - 思想：分治，拆分到单个元素再两两合并
  - 时间：O(n log n)，稳定
  - 考点：归并过程、逆序对统计

### （3）桶排序、计数排序（了解）
- **计数排序**
  - 思想：用数组统计频次，再还原
  - 适用：范围小的整数排序
- **桶排序**
  - 思想：数据分到若干桶，桶内排序再合并

## 2. 二分查找
### （1）标准二分
- 有序数组中查找某个值是否存在
- 模板：左右指针，mid = (l + r) // 2，循环条件 l ≤ r

### （2）边界查找（左边界、右边界）
- **左边界**：第一个 ≥ target / 第一个 = target
- **右边界**：最后一个 ≤ target / 最后一个 = target
- 面试高频：旋转数组查找、重复元素边界

### （3）二分答案
- 不直接搜元素，而是**猜答案**，验证是否可行
- 典型题：最小最大距离、分割数组最大值、木材切割

---

## 3. 双指针
### （1）快慢指针
- 一快一慢，步长不同
- 典型：链表判环、找环入口、找链表中点

### （2）左右对撞指针
- 左右两端向中间逼近
- 典型：两数之和Ⅱ、盛最多水、反转字符串、回文串

### （3）滑动窗口入门
- 维护一个窗口 [left, right]，动态伸缩
- 典型：无重复字符最长子串、长度最小子数组

---

# 三、中级算法与数据结构（重点）
## 1. 树结构
### （1）二叉树
- **前序、中序、后序遍历**
  - 递归版：必须秒写
  - 非递归版：用栈模拟，面试高频
- **层序遍历**
  - 用队列 BFS
  - 输出按层打印、右视图、锯齿遍历

### （2）最近公共祖先 LCA
- 二叉树普通版
- 二叉搜索树版（利用大小性质）

### （3）二叉搜索树 BST
- 性质：左 < 根 < 右
- 中序遍历 = 递增序列
- 操作：增删查、验证BST、恢复BST、第k小元素

### （4）堆
- 大顶堆、小顶堆结构
- 上浮、下沉操作
- 堆排序实现
- TopK 问题（海量数据高频）

---

## 2. 图论基础
### （1）图的表示
- 邻接矩阵：适合稠密图
- 邻接表：适合稀疏图（最常用）

### （2）DFS 深度优先搜索
- 递归/栈实现
- 应用：岛屿数量、连通块、路径存在性、回溯类图题

### （3）BFS 广度优先搜索
- 队列实现
- 应用：最短路径（无权图）、层序、扩散类问题

### （4）拓扑排序
- 有向无环图 DAG
- 入度表 + BFS
- 应用：课程安排、编译依赖、任务调度

---

## 3. 递归与回溯
### （1）全排列、子集、组合
- 排列：有顺序，可重复/不可重复
- 子集：选或不选
- 组合：不考虑顺序的选取

### （2）剪枝优化
- 提前排除不可能分支，大幅提速
- 典型：组合总和、N皇后、解数独

---

## 4. 动态规划入门
### （1）基础模型
- 斐波那契、爬楼梯
- 打家劫舍
- 最大子数组和（Kadane算法）

### （2）背包问题雏形
- **01背包**：物品只能选一次
- **完全背包**：物品可无限选
- 掌握：状态定义、转移方程、空间优化（一维DP）

# 四、进阶算法（面试 / 竞赛常用）
## 1. 动态规划进阶
### （1）线性 DP
- 最长上升子序列 LIS
- 最长公共子序列 LCS
- 编辑距离

### （2）区间 DP
- 在区间上枚举分割点
- 典型：戳气球、矩阵链乘、石子合并

### （3）状态压缩 DP
- 用二进制表示状态
- 典型：旅行商 TSP、集合覆盖、棋盘类状压

### （4）树形 DP
- 在树上做 DP，子树信息向上传递
- 典型：打家劫舍Ⅲ、树的直径、树的最大独立集

---

## 2. 高级数据结构
### （1）并查集 Union-Find
- 查找 + 合并
- 路径压缩、按秩合并
- 应用：朋友圈、连通图、岛屿合并、等式判定

### （2）单调栈、单调队列
- **单调栈**：找下一个更大/更小元素
- **单调队列**：滑动窗口最大值
- P1892 

### （3）前缀和、差分
- 前缀和：快速区间和
- 差分：区间批量修改
- 二维前缀和、二维差分

### （4）线段树、树状数组（进阶可选）
- 树状数组：单点更新、区间查询
- 线段树：区间更新、区间查询、更通用

---

## 3. 图论进阶
### （1）最短路径
- **Dijkstra**：单源无负权，优先队列优化
- **Bellman-Ford**：可处理负权、判负环
- **Floyd-Warshall**：多源，O(n³)，代码短

### （2）最小生成树
- **Kruskal**：边排序 + 并查集
- **Prim**：类似Dijkstra，点扩展

### （3）二分图匹配
- 判断二分图（染色法）
- 最大匹配：匈牙利算法

---

## 4. 字符串算法
### （1）KMP
- 解决：strStr()，模式串匹配
- 核心：next 数组，避免回退

### （2）字符串哈希
- 快速判断子串相等
- 应用：最长回文子串、重复子串

### （3）字典树 Trie
- 前缀树，存储字符串
- 应用：单词搜索、前缀统计、自动补全

---

### （3）素数筛

### （4）排列组合、概率期望

# 最简高效学习路线（详细版）
1. **复杂度分析**

5. **回溯 → 贪心**

6. **DP 入门 → 图论基础**
   简单DP → 建图、DFS/BFS、拓扑排序

7. **并查集、单调栈、字符串算法**
   高频数据结构 + 字符串匹配

8. **DP 进阶 + 图论进阶**
   区间/状压/树形DP + 最短路/最小生成树

9. **数学 + 高阶拓展**
   数论、组合、概率、竞赛进阶算法
